#  Ramadan Mystery Challenge

# Live Demo:
https://ramadan-mystery-challenge.vercel.app/

An interactive Ramadan-themed puzzle game where participants search for a hidden secret key to unlock the lantern and win rewards.

##  Features

*  Secure secret key validation using SHA-256 hashing.
*  Leaderboard system.
*  Anti-brute-force protection.
*  Interactive UI with animations and sound effects.
*  Ramadan-themed experience.
*  Built with Flask and JavaScript.
*  Deployable on Vercel.

##  Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript
* JSON

##  Project Structure

```
app.py
templates/
static/
winners.json
requirements.txt
vercel.json
```

##  Gameplay

1. Find the hidden secret key.
2. Enter the key on the main page.
3. If the key is correct, the lantern unlocks.
4. Winners receive points and rewards.
